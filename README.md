# SQL Course for Backend Engineering

A comprehensive collection of SQL scripts and practice exercises focused on backend development logic. This repository documents my journey through mastering data extraction, transformation, and relational database management.

## 🛠 Topics Covered

### 1. Data Cleaning & Manipulation
* **String Extraction:** Using functions like `LEN`, `LEFT`, `RIGHT`, `UPPER`, `LOWER`, and `TRIM` to sanitize user data.
* **Date & Time:** Advanced operations including `DATE_ADD`, `DATE_DIFF`, and part extraction for time-series analysis or filtering.

### 2. Complex Queries & Logic
* **Joins:** Mastering `INNER`, `LEFT`, `RIGHT`, and `FULL` joins to combine datasets effectively.
* **Aggregations:** Grouping data to extract meaningful business insights.
* **Set Operators:** Using `UNION`, `INTERSECT`, and `EXCEPT` for dataset comparisons.

### 3. Database Management
* Table creation (`CREATE TABLE`), schema design, and query optimization for backend performance.

## 🚀 Why this Repo?
As a Backend Engineering student focusing on **Python & FastAPI**, I use this repository to maintain a high level of proficiency in SQL—the backbone of any robust backend system. These queries are designed to be efficient, readable, and ready for integration with ORMs or raw SQL drivers.

## 📁 Project Structure
- `_7_Row_Level_Functions/`: Single-row transformations (Strings, Math).
- `DATE_AND_TIME/`: Advanced temporal logic and filtering.
- `Combining_Data_Topic/`: Relational logic and Join implementations.
